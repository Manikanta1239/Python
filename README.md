## 🚀 Quick Start

1. **Clone this repository:**
```bash
git clone <repository-url>
cd Python
```

2. **Set up virtual environment:**
```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
# or
venv\Scripts\activate     # Windows
```
